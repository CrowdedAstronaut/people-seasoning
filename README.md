## About This Project

This site was created using GatsbyJS and features full CRUD functionality via graphql using the NetlifyCMS.

### Focus

The goal of this site is to keep it as lightweight and accessible as possible. This site was built using:

- Mobile First Methodology
- Pixel Perfect Design Implementation
- Semantic naming conventions
- Flexbox
- CSS Grid
